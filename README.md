# bandcamp-fan-hider

A simple chrome extension to hide select fans you don't want to see.
All stored on your computer with syncing enabled.

## Installation

Go to `chrome://extensions/`, switch to developper mode and press `Load Unpacked`. Select the `bandcamp-fan-hider`.

## Usage

- Add the name of the fan account you want to hide in the extension popup.
- Remove any name using the icon on the right of it.
- You can also clean your whole list using the clean storage button.

## Current limitations

- Only the initial set of fan avatars will be removed when the page loads.
- There is a really short period of time during which the avatar is still visible.
